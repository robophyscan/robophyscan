Robotic manipulation requires object assets that are not only visually realistic, but also physically grounded, reproducible as real instances, and executable in contact-rich simulation. Existing real-scanned datasets provide faithful geometry and appearance, yet typically stop short of releasing simulator-ready interaction assets. Conversely, articulated simulation resources provide motion structure and interaction behavior, but are rarely built from large-scale real-scanned, purchasable real-world objects. Thus, we introduce **RoboPhyScan** , the first large-scale real-scanned sim-ready object dataset that turns real-world objects into reproducible manipulation assets. RoboPhyScan contains 2,545 purchasable everyday objects across 42 categories. **Each asset aligns high-fidelity scanned geometry, texture and PBR materials, articulated structure, precise physical properties, and manipulation-oriented annotations covering part pose, affordance, and grasp pose.** To build these assets at scale, we develop an efficient fabrication pipeline that combines consumer-grade geometry and appearance acquisition, standardized sim-ready asset fabrication, and concept-based manipulation knowledge annotation. Additionally we conduct a full evaluation of asset quality and downstream object manipulation. RoboPhyScan is designed as a community resource for simulation, evaluation, and reproducible robotics research grounded in real, purchasable objects.
